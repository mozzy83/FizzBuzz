The Code for FizzBuzz.

                        
                            function getValues(){
                                let fizzValue = document.getElementById("fizzValue").value;
                                let buzzValue = document.getElementById("buzzValue").value;
                            
                                fizzValue = parseInt(fizzValue);
                                buzzValue = parseInt(buzzValue);
                            
                                if (Number.isInteger(fizzValue) && Number.isInteger(buzzValue)){
                                    let fbArray = fizzBuzz(fizzValue, buzzValue);
                                    displayData(fbArray);
                                }else{
                                    alert("You must enter integers");
                                }
                            }
                            
                            function fizzBuzz(fizzValue, buzzValue){
                                let returnArray = [];
                            
                                for (let i = 1; i <= 100; i++) {        
                                    if(i % fizzValue == 0 && i % buzzValue == 0){
                                        returnArray.push("FizzBuzz");
                                    }else if(i % fizzValue == 0){
                                        returnArray.push("Fizz");
                                    }else if(i % buzzValue == 0){
                                        returnArray.push("Buzz")
                                    }else{
                                        returnArray.push(i);
                                    }
                                }
                                return returnArray;
                            }
                            
                            function displayData(fbArray){
                                let tableBody = document.getElementById("results");
                                let templateRow = document.getElementById("fbTemplate");
                            
                                tableBody.innerHTML = "";
                            
                                for (let index = 0; index < fbArray.length; index+=5) {
                                    //bring in a copy of template content
                                    let tableRow = document.importNode(templateRow.content, true);
                                    //create an array of all "td" elements in fragment(tableRow)
                                    let rowCols = tableRow.querySelectorAll("td");
                                    //populate textContent of array items
                                    //assign a class for css styling
                                    rowCols[0].textContent = fbArray[index];
                                    rowCols[0].classList.add(fbArray[index]);
                                    rowCols[1].textContent = fbArray[index+1];
                                    rowCols[1].classList.add(fbArray[index+1]);
                                    rowCols[2].textContent = fbArray[index+2];
                                    rowCols[2].classList.add(fbArray[index+2]);
                                    rowCols[3].textContent = fbArray[index+3];
                                    rowCols[3].classList.add(fbArray[index+3]);
                                    rowCols[4].textContent = fbArray[index+4];
                                    rowCols[4].classList.add(fbArray[index+4]);
                            
                                    tableBody.appendChild(tableRow);
                                }
                            }

The code is structured in three main functions.

getValues()

The first function starts by retrieving the Fizz and Buzz values entered by the user before parsing them into integers. It does a validation to make sure they are integers before sending them to our second function as input parameters...

fizzBuzz()

The second main function starts by declaring an empty array. It then uses a 'for loop' to populate that array with values starting at 1 and going to 100. As it populates the array it looks to see if the current value is a multiple of the fizz number, the buzz number, or both. If it is then the value is put into the array accordingly. The array is then returned where it can be used to call our final function...

displayData()

The final function first gives us handles to the table that we want to use to display our results as well as the template for the rows that we need to populate. It then ensures that our table does not currently contain any data before iterating through the array that we populated in fizzBuzz().

Each time the 'for loop' iterates it creates a handle to just the five 'td' elements in our table row template. It then creates an array with them. The content of the five 'td' columns for a single row is then populated with the value from the appropriate array index. The array value is also set as a class for each 'td' for CSS styling. The fully populated table row template is then appended to the table body before the next iteration is performed.
